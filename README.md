# 加速世界 - 时间膨胀时钟

## 项目介绍

加速世界是一个基于时间膨胀概念的时钟工具，可以将一天的24小时按照指定的倍率进行扩展，实现"一天当作几天用"的效果。该工具支持命令行界面和图形界面两种运行模式，提供实时的时间对比显示。

## 文件结构与功能

### 1. accelworld.py - 主程序入口

**作用**：程序的统一入口点，根据命令行参数选择运行命令行界面或图形界面。

**核心函数**：

- `main()`：主程序入口函数
  - 解析命令行参数
  - 根据参数数量选择运行CLI或GUI模式
  - 传递加速倍率参数给相应界面

**使用方法**：
```bash
# 运行命令行界面，指定加速倍率
python accelworld.py --rate 2.5
python accelworld.py -R 3.0

# 运行图形界面
python accelworld.py
```

### 2. accelworld_calc.py - 核心计算模块

**作用**：实现时间膨胀的核心计算逻辑，负责标准时间到加速时间的转换。

**核心类与函数**：

- `AcceleratedWorld` 类：时间膨胀计算的核心类
  - `__init__(time_dilation_rate)`：初始化时间膨胀倍率
  - `get_custom_time()`：计算并返回标准时间和加速时间
  - `run_live_clock()`：运行实时更新的加速时钟

- `main_cli()`：命令行界面主函数
  - 解析命令行参数
  - 初始化`AcceleratedWorld`实例
  - 启动实时时钟

**计算公式**：

1. **自定义一天小时数**：
   ```python
   custom_hours_per_day = 24 * time_dilation_rate
   ```

2. **标准时间总秒数**（包含微秒精度）：
   ```python
   total_seconds = current_hour * 3600 + current_minute * 60 + current_second + current_microsecond / 1e6
   ```

3. **加速时间总秒数**：
   ```python
   custom_total_seconds = total_seconds * time_dilation_rate
   ```

4. **加速后当天剩余小时数**：
   ```python
   total_custom_seconds_per_day = expanded_hours_per_day * 3600
   remaining_seconds = total_custom_seconds_per_day - custom_total_seconds % total_custom_seconds_per_day
   remaining_hours = remaining_seconds / 3600
   ```

### 3. accelworld_gui.py - 图形界面模块

**作用**：实现直观的图形用户界面，提供实时时间显示和加速倍率调整功能。

**核心类与函数**：

- `AcceleratedWorldGUI` 类：图形界面的核心类
  - `__init__(root)`：初始化图形界面
  - `setup_ui()`：设置界面布局和组件
  - `on_slider_change(value)`：滑杆值变化时的回调函数
  - `apply_acceleration()`：应用加速倍率的函数
  - `update_clock()`：实时更新时钟显示的函数

- `main_gui()`：图形界面主函数
  - 创建Tkinter根窗口
  - 初始化`AcceleratedWorldGUI`实例
  - 启动主事件循环

**布局特点**：
- 实时时钟显示区域：显示标准时间和加速时间
- 加速参数显示区域：显示加速后一天的小时数、加速倍率和剩余小时数
- 日期显示区域：显示中文格式的当前日期
- 加速设置区域：提供加速倍率输入框、滑杆和应用按钮

### 4. accelworld_date.py - 日期与农历处理模块

**作用**：提供中文日期格式转换、农历信息计算和节日显示功能。

**核心函数**：

- `get_chinese_date(now)`：将datetime对象转换为中文日期格式，包含农历信息和节日
  - 参数：`now` - datetime对象
  - 返回：中文日期字符串（格式：YYYY年MM月DD日 星期X 农历XX年XX月XX日 节日）

**主要功能**：
- 公历到农历的转换，支持天干地支年、生肖显示
- 月相和时辰信息计算
- 节日显示（包括法定假日、传统节日和自定义节日如圣诞节等）
- 使用lunar-python库获取农历核心信息
- 使用chinese-calendar库判断法定假日
- 自定义节日补充机制，确保所有重要节日都能正确显示

## 技术特点

1. **模块化设计**：核心计算、图形界面、日期处理分离，便于维护和扩展
2. **双界面支持**：同时支持命令行和图形界面，满足不同使用场景
3. **实时更新**：图形界面每100毫秒更新一次，确保时间显示的实时性
4. **灵活的加速倍率**：支持1.0到20.0之间的任意加速倍率
5. **中文日期与农历支持**：提供友好的中文日期显示，包括农历信息
6. **丰富的农历功能**：支持天干地支年、生肖、月相、时辰等传统历法信息
7. **完整的节日显示**：结合第三方库和自定义节日列表，确保所有重要节日都能正确显示
8. **第三方库集成**：使用lunar-python和chinese-calendar实现专业的农历计算和节日判断

## 使用示例

### 命令行界面

```bash
python accelworld.py -R 2.0

=== 加速世界 | 时间膨胀倍率2.0倍 | 一天48小时制实时时钟 ===
按 Ctrl+C 退出

标准时间：2025-12-25 12:00:00 | 自定义时间：24:00:00 | 膨胀倍率：200% | 一天小时数：48.00小时 | 当天剩余：24.00小时
```

### 图形界面

1. 运行`python accelworld.py`启动图形界面
2. 在"加速设置"区域调整加速倍率（可通过输入框或滑杆）
3. 点击"应用加速"按钮应用新的加速倍率
4. 观察实时更新的标准时间和加速时间

## 注意事项

1. 加速倍率必须大于1.0，否则程序会抛出错误
2. 最大加速倍率为20.0，避免数值过大导致计算问题
3. 图形界面需要Tkinter库支持，请确保已安装
4. 程序使用系统当前时间进行计算，确保系统时间准确
